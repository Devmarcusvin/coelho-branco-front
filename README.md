# Coelho Branco Front

Frontend da aplicação Coelho Branco desenvolvido com Next.js, TypeScript e TailwindCSS.

## 🚀 Tecnologias utilizadas

* Next.js
* React
* TypeScript
* TailwindCSS

## 📁 Estrutura do projeto

```bash
src/
public/
```

## ▶️ Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/Devmarcusvin/coelho-branco-front.git
```

Acesse a pasta do projeto:

```bash
cd coelho-branco-front
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

A aplicação estará disponível em:

```bash
http://localhost:3000
```

## 📌 Objetivo

Projeto desenvolvido para fins acadêmicos e aprendizado de desenvolvimento frontend moderno utilizando Next.js e TypeScript.
